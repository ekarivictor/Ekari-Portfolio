Managing work.exe (Portfolio Items)

To safely add a new portfolio item to the work.exe section without breaking the grid layout:

  1. Create a new HTML file for the case study (e.g. my-new-project.html). You can duplicate an existing one like zacra.html as a template.
  2. Open index.html and find the <div class="grid"> under the id="work-grid".
  3. Copy one of the existing <a href="..." class="grid-item"> blocks and paste it inside the grid.
  4. Change the href, <img src="...">, title, and tags to match your new project.

No databases are needed for work.exe! It keeps your site blazing fast.
